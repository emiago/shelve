Overview

Intro

Shelve is service that allows cross platform p2p file sharing using Webrtc technology. It is mainly designed to be used with command line interface, but it has also web interface for easier navigation.

Install

Installing is easy as it comes as single binary for server and clients.

Usage

Using simple interface allows you to share files/directories, make copies, add files to remote path, explore, search …
