Install

Install

Installing is easy as it comes as single binary for server and clients.

Install server

Easiest way run as docker:

docker run -d --net=host emiraganov/shelve:latest

or installing binary:

wget <url> -o /usr/sbin
shelved -l /var/log/syslog &

This will run shelved service on which shelve clients will use for signaling and sharing some metadata. For metadata check below.

HTTPS

To run service as https some env variables must be set before running shelved.

SERVER_TLS=true
SERVER_TLS_KEY=<location of key>
SERVER_TLS_CRT=<location of crt>

Metadata

Server will not hold or store any data about files content. All file sharing will be done p2p. Metadata about client sharing is active as long client is connected to server.

Only metadata that server holds (In Memory) is:

  • Active shared files from client and stats about files like name, type, size, modified time.
  • Hostname and remote IP are used for identification.
