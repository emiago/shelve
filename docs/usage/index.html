Usage

Usage:
  publish		- Publish current directory/file
  peers			- List of active peers
  cp 			  - Copy files/folders between a remote peer and the local filesystem
  ls			  - List peer file tree
  search 		- Search file across many peers

Publishing file/directory

shelve publish /path/of/your/dir /another/path ...

Check does your share appear with web client http://<yourdomain>/ or run command in other terminal:

shelve peers  # This will get you list of peers

To make sharing permanent check out Client Configuration

Exploring shared files

With web client this is just like using your file explorer. Using cli can be done

shelve ls <peerid>:/path

Search

Client allows you to search file across all peers. This is useful when you need to find specific file location.

shelve search <query>
